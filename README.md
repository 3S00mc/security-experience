# Security Portfolio – Pedro Luiz Costa

Portfólio técnico com foco em Segurança da Informação, Pentest Web e API, Cloud Security e fundamentos de Blue Team.  
Este repositório reúne projetos práticos, laboratórios documentados, scripts de apoio e modelos de relatório utilizados para estudo e simulações realistas de ambientes corporativos.

## Áreas de Atuação
- Pentest Web e APIs (OWASP Top 10)
- Análise de vulnerabilidades
- Segurança em Cloud (AWS e Azure)
- Enumeração e exploração controlada
- Documentação técnica e mitigação

## Metodologia
- OWASP Top 10
- MITRE ATT&CK
- Boas práticas de mercado
- Abordagem manual complementada por ferramentas

## Conteúdo do Repositório
- Labs documentados com evidências
- Write-ups técnicos
- Scripts auxiliares
- Checklists de segurança
- Modelos de relatório profissional
